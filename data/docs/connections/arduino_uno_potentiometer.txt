Board: Arduino Uno
Component: Потенциометр 10 кОм
Logic level: 5V

Wiring:
  Потенциометр левая нога  -> GND
  Потенциометр средняя нога -> A0 (сигнал)
  Потенциометр правая нога  -> 5V

Notes:
  - Можно подключать к любому аналоговому пину (A0-A5).
  - Крайние ноги можно поменять местами — изменится только направление вращения.
  - Не использовать без подключения обеих крайних ног — пин будет "плавать".
  - analogRead() на Uno возвращает диапазон 0-1023 (10 бит).

Code snippet:
  const int POT_PIN = A0;

  void setup() {
    Serial.begin(9600);
  }

  void loop() {
    int value = analogRead(POT_PIN);  // 0-1023
    Serial.println(value);
    delay(200);
  }
