# Breadboard

Category:
Prototyping / Electronics

Difficulty:
Beginner


## What is it?

A breadboard is a tool used to build temporary electronic circuits without soldering.

It allows components and wires to be connected quickly.


## Internal connections


Power rails:

Usually located on the sides.

Used for:

+5V

GND



Main area:

Holes are connected in groups.

Components placed in the same connected row share electrical connection.


## Common usage


Arduino:

5V → breadboard power rail

GND → breadboard ground rail


Components:

Inserted into breadboard rows.

Connected using jumper wires.



## Example projects

- LED circuits
- Sensor testing
- Arduino prototypes
- Learning electronics


## Common mistakes

- Not checking internal connections
- Connecting power rails incorrectly
- Placing components in the same connected row accidentally


## Safety notes

Always disconnect power before changing wiring.
